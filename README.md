# Cplorer
 Little Explorer App coded in C

## Build Instructions

### Windows
#### Requirements:
- MinGW-w64
- `make`

#### Steps:
```sh
make
```
This will generate `Cplorer.exe` in the current directory.

To clean up object files, run:
```sh
make clean
```

### Linux
#### Requirements:
- GCC
- Make

#### Steps:
```sh
make
```

**Note:** On Linux, the application may not support icons natively.
